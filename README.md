# Sentiment Analysis of Product Reviews

### Files included:
- SentimentAnalysis.ipynb (Jupyter Notebook)
- reviews.csv (Dataset)
- Sentiment_Analysis_Project_Report.pdf (Project Report)

### Tools used:
- Python
- Pandas
- TextBlob
- NLTK
- Matplotlib
